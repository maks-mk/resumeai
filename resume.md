## Проект для автоматизации с использованием ИИ на Python
**Smart AI Agent (pet-проект)** — https://github.com/maks-mk/simple-ai-agent  
Автономный CLI-агент на Python с поддержкой инструментов, самокоррекции и интеграции через Model Context Protocol (MCP).

- Спроектировал архитектуру «LLM как мозг, инструменты как исполнители», где агент самостоятельно планирует шаги, выбирает инструменты и корректирует стратегию при ошибках.
- Реализовал граф состояний на базе LangGraph с поддержкой циклов reasoning → tool call → validation → retry.
- Интеграция с файловой системой, веб-поиском и внешними MCP-серверами без изменения кода.
- Поддержка долгосрочной памяти, защиты от зацикливания и небезопасных действий, автоматическое восстановление при сбоях API.
- Асинхронная архитектура, стриминговый вывод в CLI (Rich UI), модульный дизайн и гибкая конфигурация через .env и mcp.json.
